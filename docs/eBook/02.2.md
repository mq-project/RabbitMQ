centos上面的安装
===================

rpm下载地址
* http://www.rabbitmq.com/install-rpm.html
* http://www.rabbitmq.com/releases/erlang/


yum install unixODBC unixODBC-devel wxBase wxGTK SDL wxGTK-gl 


rpm -ivh erlang-
yum install rabbitmq-   (自动解决依赖socat缺少的情况)


启动
/etc/init.d/rabbitmq-server start


如果发现错误，安装依赖Requires: socat 。
更新源wget Cno-cache http://www.convirture.com/repos/definitions/rhel/6.x/convirt.repo -O /etc/yum.repos.d/convirt.repo
yum install socat 



启动 /sbin/service rabbitmq-server start


管理界面
rabbitmq-plugins enable rabbitmq_management

http://localhost:15672  guest guest

增加远程访问
```
rabbitmqctl add_user admin admin
rabbitmqctl  set_user_tags  admin  administrator
rabbitmqctl set_permissions -p "/" admin ".*" ".*" ".*"
rabbitmqctl list_users
/etc/init.d/rabbitmq-server restart
```
